# CSI3125 Lab 1 | Surveys

The first lab deliverable for the CSI3125 UI course.

## Description

Simple form design using HTML, CSS and Gatsby

## Getting Started

### Dependencies

* Gatsby-cli

### Installing

* Install gatsby-cli
```
npm install -g gatsby-cli
```

### Executing program

* Spin up development server
```
gatsby develop
```
